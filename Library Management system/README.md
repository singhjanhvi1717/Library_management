# Library Management System

A Python-based library management system built with Object-Oriented Programming principles. Useful for learning OOP concepts like classes, encapsulation, and file handling.

---

## Features

- Add, remove, and view books
- Register and manage members
- Borrow and return books with due dates
- Track book availability in real-time
- File persistence (save/load data using JSON)
- Member class (coming soon)
- Library coordination class (coming soon)
