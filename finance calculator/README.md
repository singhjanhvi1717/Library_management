# Finance Calculator
This project is a Python-based finance calculator to help track expenses, calculate interest, and more.

## Features
- Simple CLI interface
- Calculates simple and compound interest,loan ,EMI and may more.
- Tracks expenses

“The project is designed in a modular, menu-driven way using functions. A loop allows multiple calculations per session. Input validation ensures robust interaction, and each calculation is handled in a separate function for clarity and reusability.”
